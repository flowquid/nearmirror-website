Keyboard Shortcuts

NearMirror supports keyboard shortcuts while the mirror window is focused. Most shortcuts use a modifier key (shown as MOD below).

Modifier Key​

The default modifier key depends on your operating system:

  • macOS — Right Alt (⌥)
  • Windows / Linux — Right Alt or Left Super (Windows key)

You can change the modifier key in Input Settings → Shortcut Mod.

Shortcuts Reference​

Window​

ActionShortcut
Toggle fullscreenMOD + F
Resize window to device resolution (1:1)MOD + G
Resize window (remove black borders)MOD + W
Enable / disable FPS counter overlayMOD + I

Display Rotation​

ActionShortcut
Rotate display leftMOD + ←
Rotate display rightMOD + →

Device Navigation​

ActionShortcut
HOME buttonMOD + H
BACK buttonMOD + B
APP SWITCH (recent apps)MOD + S
MENUMOD + M

Power & Screen​

ActionShortcut
Turn device screen on / offMOD + P
Power off deviceMOD + P (hold)

Notifications & Quick Settings​

ActionShortcut
Expand notification panelMOD + N
Expand quick settings panelMOD + N + N
Collapse panelsMOD + N (when panels are open)

Clipboard​

ActionShortcut
Copy device clipboard → desktopMOD + C
Paste desktop clipboard → deviceMOD + V
Manually sync clipboardsMOD + Shift + V

Touch Simulation​

ActionShortcut
Pinch-to-zoomHold Ctrl, then click and drag
Tilt (two-finger tilt gesture)Hold Shift, then click and drag

Tips​

  • Shortcuts only work when the mirror window is focused — click on it first if shortcuts don't respond
  • The MOD key can be pressed on either side of the keyboard unless you specify a side in Input Settings
